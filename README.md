# Bachat – Smart Price Tracking & UX Design Platform

A user-centered smart price tracking platform designed to help users monitor product prices, manage watchlists, receive target price alerts, and improve shopping efficiency through automated notifications.

This project was developed as part of a Human-Computer Interaction (HCI) course and follows User-Centered Design (UCD) principles, usability evaluation, and iterative design methodologies.

## Project Overview

Bachat is a smart shopping assistant that reduces the effort required to manually track product prices across online marketplaces.

The platform enables users to:

- Track products of interest
- Set target price alerts
- Manage personalized watchlists
- Receive automated notifications
- Monitor price changes efficiently

The design focuses on usability, learnability, navigation efficiency, and user experience optimization.
